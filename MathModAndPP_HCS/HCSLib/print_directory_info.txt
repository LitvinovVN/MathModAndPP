Cкрипт на Python, который рекурсивно отправляет на печать все файлы из указанного каталога
pip install pywin32 Pillow

Использование:
Базовое использование (печать всех файлов на принтер по умолчанию):
python print_directory.py C:\MyFolder

С указанием конкретного принтера:
python print_directory.py C:\MyFolder "Имя принтера"

Просмотр списка доступных принтеров:
python print_directory.py

Особенности скрипта:
Рекурсивный обход: Обрабатывает все вложенные папки
Поддержка разных типов файлов:
Текстовые файлы (.txt, .log, .py и др.)
Изображения (.jpg, .png, .bmp и др.)
PDF файлы (через системную ассоциацию)
Остальные файлы (через ассоциированные программы)

Безопасность: Запрашивает подтверждение перед началом печати

Фильтрация: Можно указать конкретные расширения файлов

Обработка ошибок: Корректно обрабатывает ошибки печати

Важные предупреждения:
Будьте осторожны! Скрипт отправит на печать ВСЕ файлы в указанной директории
Проверьте принтер перед запуском, чтобы избежать ненужной печати
PDF файлы печатаются через ассоциированную программу (должен быть установлен PDF-ридер)
Скрипт работает только на Windows из-за использования Win32 API
Рекомендуется сначала протестировать скрипт на небольшой директории с несколькими файлами.
